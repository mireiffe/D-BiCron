Cron 표현식이란?

5개의 필드로 실행 시점을 지정합니다. 각 필드는 공백으로 구분합니다.

특수 문자

*
모든 값 (매 분, 매 시, ...)
*/N
N 간격 (예: */10 → 10분마다)
N-M
범위 (예: 9-17 → 9시~17시)
N,M
목록 (예: 1,15 → 1일과 15일)

요일 값

0=일, 1=월, 2=화, 3=수, 4=목, 5=금, 6=토

자주 쓰는 예시

*/10 * * * *10분마다
0 * * * *매 시 정각
30 9 * * *매일 오전 9:30
0 9 * * 1-5평일 오전 9시
0 0 * * *매일 자정
0 9,18 * * *매일 9시, 18시
0 0 1 * *매월 1일 자정
0 0 * * 0매주 일요일 자정
